Complete How-To: Program Workflow and Upload

Follow these steps to generate your sketch and upload it to your Arduino.

1) Using This Program

  1. Click Browse and load an image, or draw in the 8×64 editor.
  2. Use Draw/Erase/Fill/Text tools to clean up pixels for better readability while spinning.
  3. Set pin values to match your real wiring (Red, Green, Blue, Hall, Commons).
  4. Click Use Drawing (8×64) (for manual art) or process the imported image.
  5. Review generated C/Arduino code in the output panel.
  6. Use Copy or Save As to export the sketch source.

2) Upload to Arduino Nano

  1. Open Arduino IDE and paste or open the generated sketch.
  2. Select Tools → Board → Arduino Nano.
  3. Select correct Tools → Processor option for your Nano bootloader.
  4. Select the Nano serial port in Tools → Port.
  5. Click Verify then Upload.
  6. Spin assembly and adjust Hall sensor/magnet gap until timing is stable.
